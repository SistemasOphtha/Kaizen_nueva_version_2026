## Descripción General del Sistema

**Kaizen Ophtha** es un sistema de gestión para el seguimiento de visitas, paneles médicos y planes de trabajo. El sistema está diseñado para:

- Gestionar visitas a paneles médicos
- Controlar justificaciones de ausencias
- Administrar usuarios y sus clasificaciones
- Generar reportes e indicadores de desempeño
- Planificar y dar seguimiento a planes de trabajo

## Requisitos del Sistema

- Navegador web moderno (Chrome, Firefox, Edge, Safari)
- Conexión a internet estable
- Credenciales de acceso proporcionadas por el administrador

## Soporte

Para soporte técnico, contacte al administrador del sistema.
