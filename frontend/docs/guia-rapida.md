# Guía Rápida de Referencia - Kaizen Ophtha

## 🚀 Inicio Rápido

### Acceso al Sistema
1. Abra el navegador e ingrese la URL del sistema
2. Ingrese su **correo** y **contraseña**
3. Haga clic en **"Iniciar Sesión"**

---

## 📊 Navegación Principal

| Módulo | Acceso | Función Principal |
|--------|--------|-------------------|
| **Tableros** | Menú lateral | Ver indicadores y reportes |
| **Consultas** | Menú lateral | Gestionar visitas y justificaciones |
| **Registros** | Menú lateral | Administrar usuarios y paneles |

---

## 📋 Tareas Comunes

### ✅ Registrar una Visita

```
Consultas > Visitas > + Nueva Visita
```

1. Seleccione **Tipo de Panel**
2. Seleccione el **Panel**
3. Ingrese **Fecha** y **Objetivo**
4. Clic en **Guardar**

### 📝 Crear una Justificación

```
Consultas > Justificaciones > + Nueva Justificación
```

1. Seleccione el **Panel**
2. Ingrese **Fecha** y **Motivo**
3. Clic en **Guardar**

### 👤 Agregar un Usuario

```
Registros > Usuarios > + Nuevo Usuario
```

1. Complete **Información Personal**
2. Complete **Información Adicional**
3. Asigne **Portafolios** (paneles)
4. Clic en **Guardar**

### 🏥 Agregar un Panel

```
Registros > Paneles > + Nuevo Panel
```

1. Complete **Información Personal** (nombre, identificación, tipo)
2. Complete **Información de Contacto**
3. Complete **Información Adicional**
4. Configure **Portafolio** si aplica
5. Clic en **Guardar**

---

## 📈 Ver Indicadores

```
Tableros > Indicadores
```

| Indicador | Significado |
|-----------|-------------|
| **Paneles Asignados** | Total de paneles a su cargo |
| **Paneles con Impacto** | Paneles visitados al menos una vez |
| **Visitas Realizadas** | Total de visitas completadas |
| **Paneles Sin Visitar** | Paneles pendientes de visita |

### Filtrar Indicadores
- Seleccione **rango de fechas**
- Seleccione **región** (opcional)
- Seleccione **usuario** (opcional)

---

## 📥 Generar Reportes

```
Tableros > Reportes
```

1. Seleccione **tipo de reporte**
2. Configure **filtros**
3. Clic en **Generar**
4. Clic en **Descargar** (Excel)

---

## 🔍 Búsqueda Rápida

En cualquier listado:
1. Use la **barra de búsqueda** superior
2. Escriba nombre, ID o criterio
3. Resultados se filtran automáticamente

## 🔧 Solución de Problemas Rápida

| Problema | Solución |
|----------|----------|
| No puedo iniciar sesión | Verifique credenciales o recupere contraseña |
| Datos no cargan | Presione F5 para actualizar |
| Error al guardar | Verifique campos obligatorios |
| Sesión expirada | Inicie sesión nuevamente |
